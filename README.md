# QuantLab

Demo comercial estática para QuantLab.

## Contenido

- Hero con propuesta de valor
- Sección de servicios
- Proceso de trabajo
- Bloque de escalado
- CTA final

## Enfoque

Base pensada para una entrega rápida, con hasta 2 revisiones incluidas y espacio para ampliar más adelante con ecommerce, IA, booking o automatizaciones si el proyecto lo necesita.

## Uso

Abre `index.html` en el navegador o sírvelo con cualquier servidor estático.

## Publicación en GitHub Pages

1. Sube los cambios a la rama `main`.
2. En GitHub, abre `Settings > Pages`.
3. En `Build and deployment`, selecciona `Deploy from a branch`.
4. Elige `Branch: main` y `Folder: / (root)`.
5. Guarda los cambios y espera a que Pages publique el sitio.

No hace falta build: el proyecto se sirve directamente como sitio estático.
